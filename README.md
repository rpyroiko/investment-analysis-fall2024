# Investment Analysis Fall 2024

## Overview
This project is dedicated to conducting a comparative stock analysis of Apple Inc. for the upcoming fall semester of 2024.

## Features
- Analysis of historical stock data
- Predictions based on statistical models

## Tech Stack
- R
- ggplot2

## Usage
To run the analysis, execute:
```bash
Rscript analysis.R
```
